## Getting Started

### Install dependencies
```sh
npm install -g @vue/cli
```

### Install packages
```sh
npm install
```

### Development
```sh
npm run serve
```

Clique em [http://localhost:8080](http://localhost:8080) para visualizar no browser.

Projeto [backend](https://github.com/Jhonatan-Pereira/node_knowledge_base_backend).
